概要
・現在不使用となっているMP回復のステータスの表示を変更し、気位を表示するようにします。ただ
　しヴァーレンの仕様上、戦闘開始直前(戦場に入る前)にイベントを実行する術が無いこと、戦場に
　入ってしまうとステータス系スキルは変更しても適用されないことから、表示できるのはプレイヤ
　ーターン中のみで、残念ながら援軍などを出す際には確認できません…。申し訳ありません。


導入方法
scriptフォルダをミドナver0.51のscriptフォルダに統合して下さい。


注意
・本体はver0.51を想定しています。
・問題が発生しても責任はとれません。自己責任でご利用下さい。報告いただければ可能な範囲で対
  応します。
