概要
内政などの各種機能を拡張・改善するmodを色々統合したものです。


詳細
/////////////////フィルター////////////////////////////////////////////////////////////
・個別メニューの内政コマンドにレベルフィルター(レベル10以下、11〜20、21〜30、31以上)
　とシステムスキルフィルター(Eカードや特殊クラスチェンジパス、各種固有能力スキルなどの
　所持、非所持で展開)を追加。
・複数条件の使用が可能に。特定クラスでレベル11以上、Eカードを所持しているオートCCOFFの
　ユニットなど絞り込みの幅が広がります。条件リセットコマンドか「全人材」展開コマンドの
　実行、またはターンを跨いだ場合に条件はリセットされます。
・フィルターを全人材オートクラスチェンジON・OFFにも適用するようにします。特定クラス・特
　定レベル帯のユニットをまとめてオートクラスチェンジ設定できるようになります。

/////////////////サブイベリプレイ///////////////////////////////////////////////////////
・サブイベントの「発生済み」の内政ユニットからそのイベントをリプレイできるようにします。

/////////////////選択オートクラスチェンジ///////////////////////////////////////////////
・個別メニューでのオートクラスチェンジの設定時にクラスチェンジ先を選べるようになります。
　ただし特殊クラスチェンジは選べません。通常の上位クラスのみです。一つも無い場合やEカー
　ドがない場合は選択肢が出ません。
・オートクラスチェンジ一括設定の項目に「系統毎のオートクラスチェンジ先指定」コマンドを追
　加します。「アウトローは全員バイクに乗れ！」とか「学生は全員水着を着ろ！」とかができま
　す。やったぜ。この効果は一括オートクラスチェンジ設定や新規加入ユニット一括オートクラス
　チェンジ設定でオートクラスチェンジされる場合も適用されます。前述のユニットごとのクラス
　チェンジ先設定をしている場合はそちらが優先されるので、例外を設定したい場合はそちらで。
　個別設定同様、特殊クラスチェンジはサポートしていません。

/////////////////クラス一覧表示/////////////////////////////////////////////////////////
・世界地図右上(槍近海9の右あたり)にクラス一覧の領地を表示します。イベント読み込みのタイミ
　ングの都合で、導入前にセーブしたデータはロード直後には出ない場合があります。問題が発生し
　ていなければターンを跨ぐと出ると思います。クラスチェンジ前に能力を確認するのにご利用下さ
　い。

/////////////////忠誠低下ONOFF・忠誠低下OFF時忠誠増減置き換え///////////////////////////
・ver0.51でノーマル以下の難易度では忠誠が低下しない設定になりましたが、これらの難易度では
　設定のオンオフを内政コマンドで切り替えられるようにしました。勢力戦略の「忠誠低下ON」ま
　たは「忠誠低下OFF」の内政ユニットから変更できます。人材プレイでは個別メニューにあります
　。
・上記の忠誠低下設定を変更していない場合(忠誠が低下しない場合)、忠誠変動をスキル「使命感」
　のレベル変動に置き換えます。この機能は起動時に出る選択肢でONOFFを変更できます。変動量は
　以下のとおりです。なお、未習得はレベル0とします。
	・忠誠増減→忠誠値5をレベル1に換算し、増減した値分レベルが増減。
	  ただし変動忠誠値が1以上5未満の場合は切り上げて5として扱う。
	・忠誠低下抑制→使命感のレベル低下量が半減。
	・忠誠低下無効→使命感のレベルが下がらない。
　使命感の性能は以下のとおりです。
	・本家のスキル「高揚感」の改変スキルです。あちらとは独立して機能します
	・前述の条件でのみレベル変動します。「高揚感」の変動時には特に変化はありません。
	・最大レベルは10、効果量はレベル×1。
　自愛や忠臣持ちが飛び抜けて受ける恩恵が大きかったりとゲームバランス的な問題があるので、今
　後何か思いついたら調整予定です。

導入方法
フィルター追加、サブイベントリプレイはこのmodに統合されました。
上記の統合元のmodのファイルは削除して下さい。ミドナ本体由来のファイル・フォルダは上書き
するのでそのままで構いません。

1.scriptフォルダをミドナver0.51のscriptフォルダに上書きコピーして下さい。本体ファイルを上書
　きすると困る場合は、変更点に「mod変更行」とコメントを入れているので検索して手動で書き移し
　てください。
3.共通処理フォルダは同作者のmodの共通処理の入ったファイルです。一番新しいmodのもので上書き
　されるようにscriptフォルダに上書きコピーして下さい。
